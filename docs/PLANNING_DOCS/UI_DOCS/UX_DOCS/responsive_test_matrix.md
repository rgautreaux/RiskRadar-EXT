# Responsive Test Matrix for RiskRadar Web

**Author:** Rebecca (Layout Lane)
**Last Updated:** 2026-03-23

## Purpose
This matrix lists all breakpoints and expected layout changes for each major region of the web app.

---

| Region         | Desktop (≥900px)         | Tablet (600-899px)         | Mobile (<600px)         |
|----------------|-------------------------|----------------------------|-------------------------|
| Topbar         | Horizontal, full nav    | Stacked, nav wraps         | Stacked, nav collapses  |
| Page Shell     | Max width 1280px, grid  | Reduced padding, grid      | Single column, min gap  |
| Panels/Cards   | 2-3 columns, large gap  | 1-2 columns, med gap       | 1 column, small gap     |
| Icon Sizes     | md-lg                   | sm-md                      | sm                      |
| Typography     | lg-xl headings          | md-lg headings             | sm-md headings          |

---

## Notes
- Test each region at all breakpoints after layout or CSS changes.
