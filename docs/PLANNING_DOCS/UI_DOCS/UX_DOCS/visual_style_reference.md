# RiskRadar Visual Style Reference

**Author:** Rebecca (Layout Lane)
**Last Updated:** 2026-03-23

## Color Palette
- Accent Coral: #ef6f51
- Accent Amber: #f2b441
- Accent Teal: #1b8a89
- Ink: #122231
- Muted: #5f6b77
- Panel: #fffaf2 (base), #fff8ee (strong)

## Typography
- Headings: Space Grotesk, sans-serif
- Body: Space Grotesk, sans-serif
- Code/Meta: IBM Plex Mono, monospace
- Font weights: 400 (regular), 500 (medium), 700 (bold)

## Iconography
- PNG icons: wireframe_icons/
- Icon sizes: icon-sm (16x16), icon-md (24x24), icon-lg (32x32), icon-xl (48x48)
- Use .icon-slot and .icon-badge for consistent backgrounds

## Example Usage
```html
<h1 class="text-xl" style="color: var(--accent-coral)">RiskRadar</h1>
<span class="icon-slot"><img src="/wireframe_icons/alerts.png" class="nav-icon" alt="Alerts"></span>
```

## Notes
- For new color or typography needs, propose additions here before using in templates.
