# Screen Reader Walkthrough Script for QA/Docs

**Author:** Rebecca (Layout Lane)
**Last Updated:** 2026-03-23

## Purpose
This script guides QA/Docs through a screen reader test of RiskRadar web pages.

---

## Steps
1. Open the page in Chrome or Firefox with NVDA or VoiceOver enabled.
2. Tab through the navigation bar. Confirm all links are announced and focus is visible.
3. Navigate to main content. Confirm headings and regions are announced in order.
4. Trigger an alert or error state. Confirm the message is announced via live region.
5. Open a modal/dialog. Confirm focus is trapped and dialog is announced.
6. Fill out a form. Confirm all fields have labels and errors are announced.
7. Check for color contrast and visible focus at all steps.

---

## Notes
- Record any missing announcements or navigation traps for follow-up.
- Use this script for each major page and after any accessibility-related change.
