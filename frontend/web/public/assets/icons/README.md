# Icons Directory

This directory contains SVG icon assets for the RiskRadar web frontend. Icons are sourced from UI_UX_STYLE_FILES/assets/svg/icons/ and should be referenced in navigation and UI components via relative paths.

- Only use icons that have been reviewed and approved.
- Add alt text for accessibility when using icons in HTML.
- Do not use icons directly in templates until reviewed by both Layout and Template Lanes.
- Optimize SVGs for web if needed.
