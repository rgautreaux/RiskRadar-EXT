# Illustrations Directory

This directory contains SVG illustration assets for the RiskRadar web frontend. Illustrations are sourced from UI_UX_STYLE_FILES/assets/svg/illustrations/ and should be referenced in UI components via relative paths.

- Only use illustrations that have been reviewed and approved.
- Add alt text for accessibility when using illustrations in HTML.
- Optimize SVGs for web if needed.
