# RiskRadar Web Frontend (CMPS 357)

This directory is reserved for the Stage 1 PHP web-app extension.

## Purpose
- Build a unique web experience distinct from the mobile app.
- Reuse the shared backend APIs from `backend/`.

## Planned Structure
- `frontend/web/views/`
- `frontend/web/components/`
- `frontend/web/services/`
- `frontend/web/public/`
- `frontend/web/config/`

## Notes
- The mobile app remains in `frontend/mobile/RiskRadar/`.
- Web and mobile frontends should both target API routes under `/api/v1`.
