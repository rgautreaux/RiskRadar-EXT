# RiskRadar Mobile App

This directory contains the existing Expo/React Native mobile frontend for RiskRadar.

## Get Started

1. Install dependencies with `npm install`.
2. Start the Expo development server with `npx expo start`.

## Notes

- This mobile client is preserved under `frontend/mobile/`.
- The CMPS 357 web-app extension is being developed separately under `frontend/web/`.
- Both frontends are expected to work from the same backend in `backend/`.

The original scaffold readme content is also preserved in `README-BECCA-COMPUTER.md`.
