# Frontend Directory

This repository uses the web frontend for the required CMPS 357 workflow while sharing the same backend in `backend/`.

## Structure

- `web/` — dedicated CMPS 357 PHP web-app extension workspace.

## Mobile Note

- `frontend/mobile/` is not required for this repository's setup path.
- If mobile files are absent, continue with backend + web workflows.
- Mobile app work is maintained in a separate repository.

## Shared Backend

The web frontend consumes the same FastAPI backend and API routes implemented in `backend/`.

## Entry Points

- See `web/README.md` for the web frontend scaffold and intended structure.
