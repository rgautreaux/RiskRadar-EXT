# wireframe_icons Directory Guide

**Purpose:**
This README documents the contents, naming conventions, and usage rules for the wireframe_icons/ directory. Maintained by Layout Lane (Rebecca).

---

## Directory Contents
- All PNG icon assets used for wireframe-accurate navigation and UI
- Reference wireframe images (e.g., RiskRadar_Web_Wireframes.png)
- Any overlays, callouts, or legend files for design documentation

---

## Naming Conventions
- Use `RiskRadar_[Region/Function]_Icon.png` for UI icons
- Use `RiskRadar_Web_Wireframes.png` for full-page wireframe reference
- Use lowercase, underscores for overlays/legends (e.g., `legend_nav.png`)

---

## Usage Rules
- Only reference icons in navigation or UI via app.css/layout.php
- Do not use icons directly in templates until reviewed by both Layout and Template Lanes
- Optimize PNGs for web (compress if >100KB)
- Add alt text for all icons when used in HTML

---

## Maintenance
- Add new icons with clear, descriptive names
- Remove unused/obsolete icons after team review
- Update this README with any new asset types or rules
